# Language Detection using Machine Learning & NLP

This project detects the language of a given text using
Natural Language Processing and Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- NLP (CountVectorizer)
- Multinomial Naive Bayes

## Features
- Detects multiple languages
- Accepts user input
- High accuracy (~95%)

## How It Works
1. Text is converted into numerical features using CountVectorizer
2. Multinomial Naive Bayes model is trained
3. Model predicts the language of the input text

## Example
Input:
我是薩蒂揚·辛格·拉傑普特

Output:
Chinese
