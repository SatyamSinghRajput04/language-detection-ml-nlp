# Language Detection using ML & NLP

This project detects the language of a given text using
Machine Learning and Natural Language Processing.

## Tech Stack
- Python
- Scikit-learn
- CountVectorizer
- Multinomial Naive Bayes

## Features
- Detects multiple languages
- User text input
- Accuracy ~95%

## Example
Input:
我是薩蒂揚·辛格·拉傑普特

Output:
Chinese
